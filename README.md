Command line interface - http://unific.net

![http://gyazo.com/9c7d373fe37393e927c6a7de81b0018e.png]

### これはなに

* [Unific](http://unific.net)のCLIインターフェースです
* コンソールでRSSリーダが読めます

### Usage

     npm install -g unific-cli
     unific-cli nikezono
